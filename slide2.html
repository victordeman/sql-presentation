CRUD Operations in SQL

CRUD stands for Create, Read, Update, Deleteâ€”the four essential operations for managing data in a database. These operations allow users to insert new records, retrieve data, modify existing records, and remove unwanted data, forming the backbone of database management.

Create: Add a New Record

INSERT INTO Sales (Product, Category, Sales_Amount)
VALUES ('Tablet', 'Electronics', 600);

Read: Retrieve Data

SELECT Product, Sales_Amount 
FROM Sales 
WHERE Category = 'Electronics';

Update: Modify Data

UPDATE Sales 
SET Sales_Amount = 900 
WHERE Product = 'Phone';

Delete: Remove Data

DELETE FROM Sales 
WHERE Product = 'Shirt';
