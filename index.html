What is SQL?

SQL (Structured Query Language) is used to query and manage databases.

It plays a key role in extracting insights from data like sales figures and customer behavior.

Tools like SQLValidator, developed by Dr. Obionwu, help learners master SQL effectively.

Basic Commands:

  • SELECT - Retrieve data
  • WHERE - Filter results
  • GROUP BY - Aggregate data

Sample Sales Table

Product Category Sales_Amount
Laptop Electronics $1500
Phone Electronics $800
Shirt Clothing $200
Jeans Clothing $300
Headphones Electronics $400

Practical SQL Example

SELECT 
    Product, 
    SUM(Sales_Amount) as Total_Sales 
FROM Sales 
GROUP BY Product 
ORDER BY Total_Sales DESC 
LIMIT 5;

Explanation:

  • Retrieves top 5 products by total sales
  • Groups sales by product and sums amounts
  • Orders results from highest to lowest sales 
SELECT 
    Product, 
    SUM(Sales_Amount) as Total_Sales 
FROM Sales 
WHERE Category = 'Electronics' 
GROUP BY Product 
ORDER BY Total_Sales DESC 
LIMIT 5;
